# Claudes

Open source packages and tools for developer productivity.

## Packages

### [@hoop71/claude-retro](./packages/claude-retro)

Automated developer retrospectives for any coding tool.

[![npm version](https://badge.fury.io/js/@hoop71%2Fclaude-retro.svg)](https://www.npmjs.com/package/@hoop71/claude-retro)

**Features:**
- Automated time tracking via hooks
- Git commit correlation
- Jira integration
- Privacy-first local storage
- Works with Claude Code, Cursor, and custom tools

## Installation

Install packages from NPM:

```bash
# Claude Retro
npm install -g @hoop71/claude-retro
```

## Development

This is a monorepo containing multiple packages.

```bash
# Clone the repository
git clone https://github.com/hoop71/claudes.git
cd claudes

# Work on a specific package
cd packages/claude-retro
npm install
npm link
```
